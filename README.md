# Mobile Robotics Coding Assignments

This repository contains coding assignments using ROS2 completed for my Mobile Robotics course.

Each folder contains the code, resources, and any relevant documentation for that assignment.

## Prerequisites
ROS2 (Jazzy)
Python
Required Python packages (see individual assignment folders)
Setup:
- Clone the repository
- colcon build
- source install/setup.bash

We are utilizing a Turtlebot for these assignments, however the first assignment can be done viturally without any hardware!
Just download the data and run the virtual sim. (I have to check with my professor if I can share the data files as it is not mine)

## Topics Covered
- Path planning
- Perception (LiDAR, Stereo RBG-depth camera, point clouds)
- Autonomous navigation

## Notes
Each assignment may have its own setup instructions.
Refer to individual folders for specific details and launch commands.
